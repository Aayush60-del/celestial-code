P-204 Hard ID: 1024

Celestial Pathfinding:
Matrix Pulse

Given an m × n grid representing a celestial navigation map, determine the minimum energy path to travel from the top-left corner to the bottom-right corner. Each cell contains a pulse value that consumes energy.

You can only move down or right. However, if a cell contains a Supernova Pulse (value < 0), you can consume it to regain energy.

Example 1

Input:

grid = [[1,3,1],[1,5,1],[4,2,1]]

Output:

7

Because the path 1→3→1→1→1 minimizes the sum.

Constraints
  • m == grid.length
  • n == grid[i].length
  • 1 <= m, n <= 200
  • 0 <= grid[i][j] <= 100
